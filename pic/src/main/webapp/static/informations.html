Questo box permette di aggiornare il calendario posto a destra. E' diviso in quattro parti che servono a:
  • avanzare di un mese ( freccia alla destra della scritta Mese )
  • retrocedere di un mese ( freccia alla sinistra della scritta Mese )
  • avanzare di una settimana ( freccia alla destra della scritta Settimana )
  • retrocedere di una settimana ( freccia alla sinistra della scritta Settimana )
  • selezionare tutte le occorrenze di un giorno della settimana premendo sull'iniziale del giorno voluto.
  • deselezionare tutte le occorrenze di un giorno della settimana premendo sull'iniziale del giorno voluto.
  • avanzare di un giorno ( freccia alla destra della scritta Giorno )
  • retrocedere di un giorno ( freccia alla sinistra della scritta Giorno )
